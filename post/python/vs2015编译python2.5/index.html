VS2015编译python2.5

6月 22, 2017 1 分钟阅读时长 Python

阅读《Python源码剖析》，在Windows环境下编译Python。
下载源码，用VS2015打开PCBuild8目录下的工程文件。 项目属性单启动项目改为python，配置属性只需生成python，pythoncore。
将make_versioninfo_d工程属性中的目标文件名修改为make_versioninfo_d。
将pythoncore工程属性中的目标文件名修改为python25_d。
向pythoncore工程中的timemodule.c文件中添加：

#if defined _MSC_VER && _MSC_VER >= 1900
#define timezone _timezone
#define tzname _tzname
#define daylight _daylight
#endif

向pythoncore工程中添加_typesmodule.c文件，该文件位于python2.5/Modules目录。
