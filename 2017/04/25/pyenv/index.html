pyenv安装使用

Posted on 2017-04-25

环境

Ubuntu 16.10

安装pyenv

安装指定版本的python

pyenv install 3.5.2

问题

到这里卡住不动了，无法从python官网下载。

解决方案

通过国内镜像下载到本地再安装。

安装依赖包，否则会出现以下问题。

安装openssl-devel

其他依赖包

开始安装后会在这里停留几分钟，耐心等待即可。

安装pyenv-virtualenv

创建虚拟环境
pyenv virtualenv 3.5.2 venv352
激活虚拟环境
pyenv activate venv352
退出虚拟环境
pyenv deactivate
